## 🔧 РАСЧЕТЫ ЗВУКОВОГО ДАВЛЕНИЯ (SPL)

### Расчет для арены

**Исходные данные:**
- Расстояние от колонки до центра арены: **6.5 м**
- Текущая конфигурация: 4 колонки по 800 Вт
- Текущий SPL: **111.8 дБ** ✓ (требуется 95 дБ)

**Рекомендуемая конфигурация:**
- **4 колонки по 1200 Вт**
- Расчетный SPL: **113.6 дБ** ✓
- Запас: **18.6 дБ**

**Формула расчета:**
```
SPL = SPL_1m + 10×log₁₀(P) - 20×log₁₀(r) + 10×log₁₀(N) + 10×log₁₀(Q/4π)
SPL = 95 + 10×log₁₀(1200) - 20×log₁₀(6.5) + 10×log₁₀(4) + 10×log₁₀(8/4π)
SPL = 95 + 30.8 - 16.5 + 6.0 - 2.9 = 113.6 дБ
```

**Зоны покрытия:**
- Зона покрытия одной колонки (90°, 10 м): **78.5 м²**
- Общая зона покрытия (4 колонки): **314.2 м²**
- Площадь арены: **132.7 м²**
- Покрытие: **236.7%** (с запасом)

### Расчет для зала

**Исходные данные:**
- Расстояние до последнего ряда: **35.0 м**
- Текущая конфигурация: 4 колонки по 500 Вт
- Текущий SPL: **95.2 дБ** ✓ (требуется 95 дБ)

**Рекомендуемая конфигурация:**
- **6 колонок по 1000 Вт**
- Расчетный SPL: **99.9 дБ** ✓
- Запас: **4.9 дБ**

**Формула расчета:**
```
SPL = 95 + 10×log₁₀(1000) - 20×log₁₀(35) + 10×log₁₀(6) + 10×log₁₀(8/4π)
SPL = 95 + 30.0 - 30.9 + 7.8 - 2.9 = 99.9 дБ
```

**Зоны покрытия:**
- Зона покрытия одной колонки (90°, 15 м): **176.7 м²**
- Общая зона покрытия (6 колонок): **1,060 м²**
- Площадь зала: **1,200 м²**
- Покрытие: **88.3%** (требуется дополнительное покрытие или корректировка)
